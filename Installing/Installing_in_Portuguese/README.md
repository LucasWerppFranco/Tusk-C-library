# Instalação

---

1 - Baixe e abra o repositório:

```bash
git clone https://github.com/LucasWerppFranco/Tusk-C-library.git
cd Tusk-C-library
```

2 - Execute o Makefile e use-o para instalar a biblioteca:

```bash
sudo make uninstall
```

---

# Desistalando biblioteca

```bash
sudo make uninstall
```
